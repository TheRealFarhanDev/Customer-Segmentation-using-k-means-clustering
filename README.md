# Customer Segmentation using K-Means Clustering

## Overview

This project uses the K-Means Clustering algorithm to segment customers based on their annual income and spending score. Customer segmentation helps businesses understand different customer groups and improve marketing strategies.

## Objectives

- Group customers with similar behavior
- Analyze customer spending patterns
- Visualize customer clusters
- Help businesses target customers effectively

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset

The dataset contains customer information such as:

- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Project Workflow

1. Import required libraries
2. Load the dataset
3. Perform data preprocessing
4. Visualize the data
5. Find optimal number of clusters using Elbow Method
6. Apply K-Means Clustering
7. Visualize the customer segments

## Installation

Clone the repository:

```bash
git clone https://github.com/TheRealFarhanDev/Customer-Segmentation-using-k-means-clustering.git
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## Running the Project

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook file and run all cells.

## Output

The project generates visualized customer clusters to identify different customer groups based on purchasing behavior.
